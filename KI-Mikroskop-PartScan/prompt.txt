**REFERENZ: SMD-WIDERSTANDS-CODES**
================================================================================

**3-stelliger Code (E24, ±5%):**
- Erste zwei Ziffern = Wert
- Dritte Ziffer = Anzahl Nullen
- Beispiel: 472 = 4700Ω = 4.7kΩ

**4-stelliger Code (E96, ±1%):**
- Erste drei Ziffern = Wert
- Vierte Ziffer = Anzahl Nullen
- Beispiel: 4702 = 47000Ω = 47kΩ

**EIA-96 Code (±1%):**
- Zwei Ziffern + Buchstabe
- Ziffern = E96-Index (01-96)
- Buchstabe = Multiplikator

| Code | Multiplikator |
|------|---------------|
| Z | 0.001 |
| Y/R | 0.01 |
| X/S | 0.1 |
| A | 1 |
| B/H | 10 |
| C | 100 |
| D | 1000 |
| E | 10000 |
| F | 100000 |

Beispiel: 01A = 100Ω, 68C = 49.9kΩ

**JIS (Japan):**
- Format: 2S + Buchstabe + Nummer
